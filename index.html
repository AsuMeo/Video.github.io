Установщик файлов Мессенджера

Это интерактивный дашборд для мобильных телефонов Android. Никаких ПК и сборщиков! Скачай файлы по отдельности прямо сейчас на телефон и загрузи в репозиторий GitHub Pages.

Инструкция по установке (с телефона):

  1. Нажми на каждую кнопку ниже, чтобы скачать 4 основных файла.
  2. Создай публичный репозиторий на GitHub (например, my-messenger).
  3. Зайди в него через мобильный браузер, нажми Add file -> Upload files и выбери скачанные файлы: index.html, design.css, auth.js, functionality.js.
  4. Перейди в настройки репозитория (Settings -> Pages), в разделе Build and deployment выбери ветку main (или master) и нажмите Save.
  5. Через пару минут твой полноценный Realtime Messenger будет работать по ссылке https://твой-логин.github.io/my-messenger/!
Разработано с использованием чистых ES модулей Solid.js без сборщиков.
