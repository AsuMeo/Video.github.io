Безопасность

Текст, имена участников и WebRTC-сигналинг шифруются в браузере алгоритмом AES-256-GCM. Звонки защищены обязательным для WebRTC шифрованием DTLS-SRTP.

Firebase всё ещё видит IP подключения, ID комнаты, случайные ID пользователей и время активности. При прямом P2P-звонке собеседник может определить ваш IP. Для сокрытия IP нужен собственный TURN-сервер и iceTransportPolicy: "relay".

Это клиентское приложение без независимого аудита. Для критичных данных используйте проверенный мессенджер.
